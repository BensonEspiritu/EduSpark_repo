# 🚀 Getting Started

Follow the steps below to run **EduSpark** on your local machine.

## Prerequisites

Make sure you have the following installed:

- XAMPP (Apache & MySQL)
- PHP 8.x (included with XAMPP)
- MySQL
- Visual Studio Code (optional, for editing)

## Installation

### 1. Clone the Repository

```bash
git clone https://github.com/BensonEspiritu/EduSpark_rep.git
```

Or download the repository as a ZIP file and extract it.

### 2. Move the Project

Copy the **EduSpark_rep** folder into your XAMPP `htdocs` directory.

**Windows**

```text
C:\xampp\htdocs\EduSpark_rep
```

### 3. Start XAMPP

Open the **XAMPP Control Panel** and start:

- ✅ Apache
- ✅ MySQL

### 4. Import the Database

1. Open **phpMyAdmin**
   ```
   http://localhost/phpmyadmin
   ```
2. Create a new database.
3. Import the provided `.sql` file located in the project (if applicable).

### 5. Run the Project

Open your browser and navigate to:

```text
http://localhost/EduSpark_rep/login.php
```

# 🛠 Editing the Project

Open the project folder in **Visual Studio Code**.

```text
File → Open Folder → EduSpark_rep
```

> **Note:** PHP files cannot be executed directly from Visual Studio Code. Always access the application through your browser using XAMPP (e.g., `http://localhost/...`).
